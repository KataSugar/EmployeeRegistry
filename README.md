## Installation
1. Clone the repository
```bash
git clone https://github.com/KataSugar/EmployeeRegistry.git
```

2. Install dependencies
```bash
cd server
npm install
cd.. 
cd client
npm install
cd..
```

3. Initialize and run Terraform
```bash
cd terraform
terraform init
terraform plan
terraform apply
```
